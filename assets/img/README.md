# Dossier images

Placer les images définitives du site dans ce dossier.

Fichiers déjà présents:
- `logo-masoda.png`: logo principal copié depuis `Pictures/logo.png`
- `event-paint-jam.jpg`: visuel de l'atelier du 21/02/2026
- `antiinflamatoire.png`: exemple partenaire nutritif afro anti-inflammatoire
- `baya_bijoux.png`: exemple partenaire bijoux de corps (Baya)
- `favicon.ico`: favicon placeholder généré à partir du logo
- `og-image.png`: image OpenGraph placeholder (1200x630)
- `placeholder-event.svg` et `placeholder-partner.svg`: images temporaires

Remplacement recommandé:
1. Déposer les nouvelles images dans `./assets/img/`
2. Garder des noms simples (ex: `atelier-love-2026.jpg`)
3. Pour remplacer l'image OG: conserver le nom `og-image.png` et un format 1200x630
4. Pour remplacer le favicon: conserver le nom `favicon.ico` (32x32 ou 64x64 recommandé)
5. Mettre à jour `./data/events.json` (`cover_image`) et les balises `<img>` des pages si nécessaire
